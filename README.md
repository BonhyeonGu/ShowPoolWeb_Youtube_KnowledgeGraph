## How to Install (Bare Metal)

해당 프로그램은 아래의 추가적 Python Module들을 필요로 합니다.

 - flask
 - neo4j
 - pymongo

또한 크롬 웹 브라우저를 기반으로 작성 되었습니다.

## How to Run (Bare Metal)

workspace 내부의 app.py로
```bash
flask run
```
을 통하여 웹 서비스를 실행할 수 있습니다.  
http://127.0.0.1:5050 으로 접속 가능합니다.
